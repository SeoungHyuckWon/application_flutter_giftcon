# My giftcon mobile-app

My app with flutter
이미지를 직접 촬영 또는 앨범을 통해서 저장.
저장은 Firestore을 활용한다.
저장 key나 저장된 기프티콘 사진의 정보, 회원 정보는 모두 firebase에서 관리한다.
